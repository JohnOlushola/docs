This API gives programmatic access to all the functionality of the platform:

* retrieve open-access asset-level and historical energy system data, free forever
* dispatch system modelling analysis jobs
* access premier research outputs by TransitionZero

## Authentication

A user account can be created [here](https://transitionzero.org/). Please [get in touch](mailto:info@transitionzero.org) if you have any questions or requests for the platform!
